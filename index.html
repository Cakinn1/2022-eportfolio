Here are some of my projects

  • Netflix Clone

    Next.js, Javascript, TypeScript, HTML and TailwindCss

    This project is a Next.Js-based Netflix clone, emulating the immersive streaming experience. Users can seamlessly browse and watch a diverse range of movies and TV shows. The interface replicates Netflix's allure with a dynamic home page, stripe payment portal, and captivating content pages. This project demonstrates my aptitude in creating an engaging streaming platform while ensuring an intuitive user journey.

  • Twitter Clone

    Next.js, JavaScript, HTML and Tailwind CSS

    This Next.js-powered project mirrors the essence of Twitter, enabling users to create posts and engage in real-time discussions. The application features a streamlined feed, user-friendly profiles, and interactive components, capturing the core functionalities of Twitter. This project showcases my ability to replicate the dynamic nature of a social media platform while delivering a seamless and engaging user interface.

  • Discord Project

    React.Js, JavaScript, HTML and Tailwind Css

    Constructed with React, this project emulates the Discord ecosystem, granting users real-time chat capabilities and server-centric interaction. The application showcases dynamic text and voice channels, private messaging, and intuitive server administration. This endeavor underscores my ability to replicate the intricate functionalities of a widely-used communication platform, while ensuring a fluid and engaging user interface

  • Ecom Library App

    React.Js, JavaScript, HTML and CSS

    Designed and developed a React-based library e-commerce application that offers versatile book sorting options, including low-to-high and high-to-low prices, as well as sorting by ratings. The app enables seamless book purchases and employs skeleton loading states for improved user experience. This project showcases my proficiency in building dynamic user interfaces, implementing complex sorting algorithms, and enhancing user engagement through thoughtful design
